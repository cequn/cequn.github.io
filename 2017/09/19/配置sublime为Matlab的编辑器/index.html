配置Sublime为Matlab的编辑器

Contents

  1. 1. 新建Build System
  2. 2. 启动Matlab进行编译

最近自己的Matlab的图形界面出现一些问题，一时间也没找到解决方法，于是就试图采用Sublime Text编辑器+Matlab命令行的环境来运行自己的程序。

  • Matlab R2015a(Liunx平台,Ubuntu 16.04)

  • Sublime 3

    新建Build System

    在Sublime中选择Tools -> Build System -> New Build System，系统会创建一个JSON文件，输入以下内容：

    命名并保存这个JSON文件到默认目录（将会出现在Build System菜单中）。

    启动Matlab进行编译

    在Build System中选择你所设置的Matlab编译名称。此时随便编辑一个.m文件，可以点击Tools -> Build或者使用快捷键Ctrl+B，这时Sublime就会打开终端并编译当前文件。 但每次Build都会产生一个新的命令行界面，每次都重新启动了一次MATLAB，很耗费时间。所以在Sublime修改程序保存后，通过已开启的控制台窗口输入脚本名来运行。
